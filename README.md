# 8-Bit Computer Project

This project is made possible courtesy of Ben Eater's Youtube series for an 8-bit computer.

My goal for this project is not only to replicate Ben's model, but to better understand the electrical/computer engineering fundamentals applied to it. This writeup will analyze the schematics in more detail, and attempt to dig just a little bit further.
