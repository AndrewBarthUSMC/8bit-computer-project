# Prerequisites

Below is a list of concepts that are recommended to know beforehand, albeit not completely comprehensive:

- Voltage (V)
- Current (A)
- Resistance ( $\Omega$ )
- Ohm's Law: V = IR
- Kirchoff's Voltage and Current Laws (KVL/KCL)
- Voltage/Current Division Principles
- Capacitance (F)
- Inductance (H)
- Comparators/Operational Amplifiers
- Transistors
- SR Latch
